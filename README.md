# Spring Boot API with MongoDB

A Spring Boot based API with basic CRUD operations using MongoDB

## Description

* Language: Java 8
* Framework: Spring Boot
* Database: MongoDB

* REST API
* MVC Architecture
* Microservices
* Spring Web
* Spring Data with MongoDB
* Spring Security
* Springfox Swagger
